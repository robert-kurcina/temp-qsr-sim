# Reference

## Quick Start Rules Summary

1.  **Game Setup**
    *   Determine Game Size (Small: 250-500 BP, 4-8 models/Side).
    *   Each player builds an Assembly using Build Points (BP).
    *   Deploy Assemblies on opposite sides of the battlefield.

2.  **Turn Sequence**
    *   Determine Initiative (roll-off or alternate).
    *   Player with Initiative activates one Ready character.
    *   Activated character gets 2 Action Points (AP).
    *   Spend AP on Actions (Move, Attack, etc.).
    *   After spending AP, mark character as Done.
    *   Continue activating characters until all are Done.
    *   End of Turn: Check for Bottle Tests if applicable.

3.  **Core Mechanic: Tests**
    *   **Opposed Tests:** Both players roll 2 Base dice + Attribute + bonuses.
    *   **Unopposed Tests:** Active player rolls vs. System player (rolls 2 Base dice + 2).
    *   **Scoring:**
        *   Modifier Die (m): 4+ = 1 success.
        *   Base Die (b): 4-5 = 1 success, 6 = 2 successes.
        *   Wild Die (w): 4-5 = 1 success, 6 = 3 successes.
    *   **Result:** Highest score wins. Active player wins ties. Difference in scores is the "cascade".

4.  **Common Actions (1 AP each unless noted)**
    *   **Move:** Move up to MOV + 2".
    *   **Close Combat Attack:** Attack target in Melee Range (Opposed CCA Test).
    *   **Range Combat Attack:** Attack target at range (Opposed RCA vs. REF Test).
    *   **Disengage:** Break from engagement (Opposed REF vs. CCA Test).
    *   **Concentrate:** +1 Wild die to another action's test.
    *   **Rally:** Remove Fear tokens (Unopposed POW Test).
    *   **Fiddle:** Interact with objects (0 AP for first, then 1 AP).

5.  **Combat Resolution**
    1.  **Hit Test:** Attacker rolls vs. Defender. If successful, proceed to Damage Test.
    2.  **Damage Test:** Attacker rolls (Weapon Damage) vs. Defender (FOR).
    3.  **Wounds:** If Damage Test passes, target receives Wounds.
        *   Number of Wounds = cascades if > current Wounds, otherwise +1 Wound.
        *   Each Wound is a Hindrance (-1m to most Tests).
        *   Wounds ≥ SIZ = Knocked-Out (KO’d).
        *   Wounds ≥ SIZ + 3 = Eliminated.
    4.  **Armor:** Reduces Wound damage by its Armor Rating (AR). Weapon's Impact (I) reduces AR.

6.  **Morale**
    *   **Fear Test:** (Unopposed POW Test) required after taking a Wound or seeing a nearby Friendly model get KO'd/Eliminated.
    *   Failing a Fear Test adds Fear tokens.
    *   Fear tokens are a Hindrance.
    *   2+ Fear tokens = Disordered (compulsory actions).
    *   4+ Fear tokens = Eliminated.

7.  **Key Modifiers (Situational)**
    *   **Cover:** Penalizes incoming attacks (-1b for Direct, -1m for Intervening).
    *   **Hindrance (Wound, Fear, Delay):** -1m to most Tests.
    *   **Point-blank:** +1m to Range Attack at half OR.
    *   **Outnumber/Assist:** Bonuses for ganging up in Close Combat.

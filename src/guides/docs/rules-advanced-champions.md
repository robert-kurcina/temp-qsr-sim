# Advanced Rules: Champions (▲)

**Status:** ⏳ **DEFERRED** — This is a **Phase 5 feature** (post-QSR campaign system).

---

## Overview

The **Champion system** enables character progression across multiple games, tracking statistics and enabling advancement.

---

## Champion Features

### Statistics Tracked

| Stat | Description |
|------|-------------|
| **RPs Scored** | Resource Points earned |
| **OMs Acquired** | Objective Markers captured |
| **VPs Earned** | Victory Points contributed |
| **Eliminations** | Enemy models eliminated |
| **Wounds Taken** | Damage sustained |

### Advancement

- **Experience Points** from missions
- **Trait Unlocks** at milestones
- **Equipment Upgrades**

---

## Implementation Requirements

### Runtime Components
- Persistent character storage
- Statistics tracking
- Advancement system
- Campaign management

# Advanced Rules: Gas, Fume & Puffs (▲)

**Status:** ⏳ **DEFERRED** - Requires core QSR implementation plus Gas trait support.

---

## Overview

Gas markers represent various atmospheric effects including smoke, fumes, and chemical clouds.

---

## Gas Types

| Type | Effect |
|------|--------|
| **Blacksmoke** | Obscuring, generated by Fire spread |
| **Steam** | Generated when Fire meets Wet terrain |
| **Caustic** | Damaging gas cloud |
| **Knockout** | Incapacitating gas |
| **Nerve** | Lethal gas cloud |
| **Poison** | Toxic gas cloud |
| **Riot** | Crowd control gas |
| **Smoke** | Standard obscuring smoke |

---

## Gas Marker Sizes

| Size | Diameter |
|------|----------|
| **Small** | 1 MU |
| **Medium** | 2 MU |
| **Large** | 3 MU |

---

## Gas Effects

### Movement

- Models **crossing Gas** may suffer effects
- **Gas disperses** over time

### Interaction with Fire

- Gas markers **never move towards Fire**
- Gas within base-diameter of new Fire **repositions away**

---

## Implementation Requirements

### Traits Needed
- **Gas:Type** — Various gas types

### Markers Required
- **Gas markers** (multiple types and sizes)

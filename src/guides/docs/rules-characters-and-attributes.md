# Characters and Attributes

Each character is represented upon the game-play area ("battlefield") using models. All Characters are represented with a name, 9 attributes, a handful of traits, and a Build Point [ BP ] cost.

Here are the attributes in the standard order of presentation, and what each is used for:

| Attribute | Description           | Which tests? Which rules?                                      |
| :-------- | :-------------------- | :------------------------------------------------------------- |
| CCA       | Close Combat Ability  | Attacker Close Combat Hit. Defender Close Combat Hit. Disengage. |
| RCA       | Range Combat Ability  | Attacker Range Combat Hit.                                     |
| REF       | Reflexes              | Defender Range Combat Hit. Disengage. Attacker Detect. Defender Detect. Reactions and React Tests. |
| INT       | Intellect             | Initiative. Magic Link.                                        |
| POW       | Willpower             | Morale. Magic Weave.                                           |
| STR       | Strength              | Throw OR, and some Bow OR. Attacker Close Combat Damage. Attacker Range Combat Damage. |
| FOR       | Fortitude             | Defender Damage. Revive.                                       |
| MOV       | Movement              | Movement rate, direction changes, Agility, React Tests.        |
| SIZ       | Size                  | Wounds, and may affect Hit Tests                               |

### Attacker vs. Defender

The Attacker is generally the character who is performing a Test against another character or model which is the Defender.

### Attribute Values

The average attribute value for a human character is 2. Each point higher roughly doubles that attribute's effectiveness.

*   The normal human attribute value range is from 1 to 3.
    *   Exceptional humans, like those most often found in pulp action novels or science-fantasy movies, are at the peak of capability and are rated at a 4.
    *   Any attribute with a value of 5 and higher for "real life" humans is in the realm of superheroes, sci-fi, or high fantasy.
*   When comparing the attribute values between any two characters, a difference of 1 is significant, 2 is outstanding, and 3 or more is often insurmountable.

### Reducing

Whenever an Attribute value is "reduced", it is lowered towards zero and no further. If an Attribute value is already below zero, reducing it will not affect it. So, reducing a value of 3 alters it to become a 2, but reducing a value of -1 has no effect and it remains as a -1.

### Dashed Attribute Values

Some archetypes have attribute values showing dashes as in '-', '=', or '≡'. These represent -1, -2, and -3 respectively.

### The SIZ Attribute

The SIZ ("size") attribute is used to represent how much physical punishment ("Damage") a character can take before becoming KO'd ("Knocked-out") or Eliminated.

*   Most adult humanoid characters are SIZ 3.
    *   SIZ increases by 1 for every doubling of a character's mass in kilograms ("Kg of mass").
    *   SIZ 1 is about 20 Kg of mass, SIZ 2 is about 50 Kg of mass, and SIZ of 3 is about 100 Kg. A SIZ 4 character is about 200 Kg, and SIZ 5 characters are about 500 Kg.
*   The largest practical model to use in battles is SIZ 6 which is about 1000 Kg; this is mostly to represent giants, horses, or very large monsters. Smallest is SIZ 1.

### Base-contact & Base-height

When the bases of two models touch, they are in "base-contact". Bases of models should not overlap those of other models especially between models of characters which are hostile to each other ("Opposing").

Having base-diameters vary with SIZ allows for models which are on smaller bases to greatly outnumber models of larger bases when in base-contact.

A model's base-height is its base-diameter if Humanoid, and Beasts are about half the normal base-height.

### Cylindrical Volume

The volume of a model is standardized as the imaginary cylinder created from its base-diameter and its base-height. This is necessary because some models are sculpted in a dynamic pose hopping over terrain, and others are sculpted leaning forward or perhaps slinking very low to the ground.

**Melee Range** — A target is within Melee Range if its volume is in base-contact with an Opposing model's volume.

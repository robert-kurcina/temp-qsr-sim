# Variant Archetypes Reference

This module provides complete reference tables for all Variant Common Archetypes available in MEST Tactics QSR.

---

## What are Variant Archetypes?

**Variant Archetypes** are standard Common archetypes (Untrained, Militia, Average, Veteran, Elite) with **one or more additional traits**.

**Notation:**
- **Primary archetype** + **Variant trait**
- Example: "Average, Brawny" = Average archetype + Brawn trait

**BP Calculation:**
```
Variant BP = Base Archetype BP + Trait BP cost
```

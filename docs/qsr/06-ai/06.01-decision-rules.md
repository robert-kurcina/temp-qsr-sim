## Overview

AI Decision Rules codify how the AI interprets and applies QSR rules for tactical decision-making. These are derived from the QSR rules but specify AI-specific behavior.

---

## Core Decision Flow

### Activation Priority
```
1. Check compulsory actions (Disordered models)
2. Evaluate available actions based on state (Free/Engaged, AP, etc.)
3. Score each action using Utility Scorer
4. Select highest utility action
5. Execute action with context
```

## Full Canonical Text

### Performing Indirect Range Combat
```
Indirect Range Attacks are Range Attack actions against a target that is a battlefield location
which is within LOF. This may or may not be within LOS. The weapon is usually a Grenade (has
the Grenade trait), or an Item with the AoE keyword.

Unless using a Natural weapon, the Active character acquires a Delay token if this is not its
first attack this Initiative.

- Announce a Ranged weapon to be used; the weapon must have an OR value. It must be a Thrown
  weapon, have the Throwable trait, or have the [Arc] trait.
  - Thrown weapons have an OR written as STR or STR + X" to indicate that is equal to the
    Attacker's STR.
  - If OR is zero it becomes 0.5 MU. If negative, disallow the Attack.
  - If the weapon has the [Reveal] trait, remove Attacker's Hidden status if in LOS.

- Announce a battlefield location within LOF to become the target.
  - Place a reminder token at the desired target location.
  - There must exist a midpoint above the battlefield between target and attacker which is no
    higher than the distance between them, with LOS to both.
```

### Resolving Indirect Range Hit Test
```
Resolve the Indirect Attack Hit Test as an Unopposed RCA Test -1 Base die per OR Multiple (ORM).
If fail, note any misses.

Apply Situational Test Modifiers such as Hindrance, Distance, Point-blank, Direct Cover, and
Intervening Cover.

Add weapon Accuracy bonus ("Acc").
```

### Scatter
```
Scatter repositions the target location by distance and direction.
- Reposition distance = misses x 1", minimum 1 MU.
- Determine direction using a Biased Scatter diagram along LOF from attacker to target.
- If the location repositions into a Wall, continue remaining distance at reflected angle.
- If it again repositions into Wall/Obstacle/Building/Vehicle, stop at contact.

Roll-down:
- On one or more misses, reposition additional distance by gravity down slopes/precipices.
- Off a precipice: 0.5 MU per 1 MU dropped, plus 1 MU per miss.
- Check slopes (rise/run), increase reposition distance as new slopes are encountered, repeat.
- Stop if a Wall or Obstacle is hit.
```

### Scrambling
```
Scrambling is a React option when a target location is attacked.
Allowed whenever performing an Indirect Range Attack, or if [Scatter] is involved.
```

### Apply Weapon Traits
```
Apply weapon traits, starting with AoE:
- If weapon does not have Frag: targets in base-contact perform an Unopposed Damage Test.
- If weapon has Frag: apply Frag Dmg only against targets that failed its Hit Test.
```

### Scatter Diagrams
```
Unbiased:
- Random direction equally across six directions.

Biased:
- Like Unbiased, but "1" points to the Active player's desired direction.
- Re-roll the first 3, 4, or 5, then re-roll the next 4.
```

### Blind Attacks
```
Indirect attacks may target LOF locations not in LOS only if:
- Spotter: a Free Attentive Ordered Friendly model in Cohesion has LOS to target.
- Known: target was in LOS at start of Attacker's Initiative.

Blind penalties:
- Blind indirect attack: -1 Wild die on Attacker Hit Test.
- If Blind and [Scatter]: use Unbiased Scatter and add one extra Wild die for distance,
  without direction re-roll.
```

## Full QSR Text

### Close Combat Hit Test
```
Close Combat — Opposed CCA Test between Attacker and Defender.
               Attacker wins ties. Apply Situational Test Modifiers.
```

### Charge Bonus
```
Charge () — Characters that move before causing an Opposing character
             to become within Melee Range may qualify for the "Charge Bonus":
             +1 Modifier die for the Attacker Close Combat Hit Test if it
             performed a Move action into base-contact with its target,
             over Clear terrain from a Free position in a relatively straight line.

Qualification — All of these conditions must be met:
  • The Move action must have cost at least 1 AP.
  • The Active character must start Free from at least its own base-diameter
    away before the distance of possibly becoming Engaged by the target
    character or having the target become within Melee Range.
  • Movement must be directly into the target and make base-contact.
    Disallow direction changes during this Action.
  • The target must not be Hidden and be within LOS at the start of the Action,
    at no more than Visibility × 3 ORM.
  • Traits or Agility can make some terrain effectively Clear.
```

### Multiple Weapons
```
Multiple Weapons () — Characters benefit from the Multiple Weapons rule
                       if their model is shown as sculpted showing this and
                       the weapons are also purchased using BP.

Qualification — Weapons in each hand must be all capable of Ranged attacks
                (being Bow, Thrown, etc.), or all capable of Melee attacks.
  • Weapons are classified as either Melee or Ranged. Melee weapons with
    the Throwable trait count as Ranged weapons, and Ranged with [Awkward]
    count as Melee.
  • Natural weapons are their own sub-classification.
  • Improvised weapons do not count for this rule.

Assessment — Characters should be clearly sculpted with the weapon "in hand"
             unless the weapon has the Conceal or Discrete traits.
  • Count each additional hand which is armed with a Ranged or Melee weapon.
  • Count each additional pair of claws or pair of fists, or additional
    independent tail, or additional independent mouth used for biting.

Benefit — When targeting the same model:
  • Each additional Ranged weapon "in hand" provides +1 Modifier die
    Attacker Range Combat Tests.
  • Each additional Melee weapon "in hand" provides +1 Modifier die
    Close Combat Tests.

Penalty — Using the same weapon in consecutive Actions during an Initiative
          penalizes -1 Modifier die Attacker Combat Tests if there are others
          of the same classification already "in hand".

Interruptions — A character that is using a specific weapon for an attack
                which is then interrupted, such as during Reacts or Passive
                Player Options, must use that weapon for defense as well.
  • At the start of an Action, if it isn't clear, both the target and the
    Attacker must specify which weapon is in Hand.
  • The choice for weapon used when interrupted affects React Actions,
    Bonus Actions, and Passive Player Options.
```

### Situational Test Modifiers (Close Combat)
```
+1m  Charge. Attacker Hit Test if moved into base-contact with target,
      over Clear terrain from a Free position at least base-diameter.
+1m  High Ground. Disengage and Hit Tests to higher model if base is above
      half of the Opposing model's base-height, and base-height is above
      the Opposing model's volume.
+1m  Size. Disengage and Close Combat Hit Tests if in base-contact, and
      Opposing model is larger by 1, 2, 5, or 10 SIZ. Each is +1m.
+1b  Defend. Defender Hit Test if target chooses "Defend".
+1w  Outnumber. Disengage and Hit Tests for 1, 2, 5, or 10 more other
      Attentive Ordered Friendly models with the same target in Melee Range
      than the Opposing model. Each is +1w.
-1m  Cornered. Disengage and Hit Tests if Engaged to the Opposing model on
      one side of this model and in base-contact on the other side with a
      wall, precipice, or degraded terrain.
-1m  Flanked. Disengage and Hit Tests if Engaged to two Opposing models
      directly on either side of this model.
-1   Overreach. Penalized -1 REF and -1 Attacker Close Combat Tests.
```

### Close Combat Bonus Actions
```
Close Combat Bonus Actions:
  • Circle  — if in Close Combat; rotate up to 180 degrees around target.
    Terrain must be Clear, and path can't be through other models.
    Keep current separation.
  • Disengage — if in Close Combat and Engaged Attentive, reposition up to
    Agility, but at least as much as the base-diameter of the larger model.
    Allow +1 MU per 3 more cascades spent.
  • Push-back  — in Close Combat; reposition target away up to the Active
    model's base-diameter. Allow +1 MU per 3 more cascades spent. Afterwards,
    allow the Active model to reposition up to base-contact with the target.
    If the target is pushed into wall, obstacle, degraded terrain, or resists
    being pushed across a ledge or off the battlefield; it receives a Delay
    token. Disallow into another character.
  • Pull-back  — if in Close Combat; reposition the Active model away from
    the target the larger model's base-diameter. Afterwards, allow the Active
    model to reposition up to base-contact with the target.
  • Reversal  — if in Close Combat, switch positions with target.
    Keep current separation.
```

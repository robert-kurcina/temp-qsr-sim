## Full QSR Text

### Damage Test
```
Damage Test — Opposed Test: Attacker Damage (STR or weapon) vs Defender FOR.
              Attacker wins ties. Apply Armor Rating reduction.
```

### Close Combat Damage
```
Close Combat Damage — Attacker uses STR for Damage Test.
                      Apply weapon Impact to reduce AR.
                      On pass, assign Wound tokens equal to cascades beyond FOR.
```

### Range Combat Damage
```
Range Combat Damage — Attacker uses weapon damage formula.
                      Apply weapon Impact to reduce AR.
                      On pass, assign Wound tokens equal to cascades beyond FOR.
```

### Armor Rating
```
Armor Rating [AR] — Reduce any Wound damage received by AR.
  • Weapon's Impact rating [I] reduces total AR by an equivalent amount.
  • If a character is attacked by a Concentrated Attack, reduce 3 AR from
    each Item Type with the Armor trait.
```

### Impact
```
Impact — Reduces target's AR by Impact value before Damage Test.
```

### Concentrated Attack
```
Concentrated Attack — Multiple attackers vs single target.
  • Reduce 3 AR from each Item Type with the Armor trait.
  • Do not reduce AR for Friendly Fire from Concentrated attacks.
```

### Wound Tokens
```
Wound Token — Assigned when Damage Test passes.
  • Each Wound token represents damage beyond FOR + AR.
  • Track on character card or with red beads.
```

### KO'd (Knocked Out)
```
KO'd — Character is KO'd when Wound tokens ≥ SIZ.
  • Place Done token and KO'd marker.
  • May be attacked (coupe-de-grace) to become Eliminated.
  • May revive with FOR Test.
```

### Eliminated
```
Eliminated — Character is Eliminated when:
  • Wound tokens ≥ SIZ × 2 (double SIZ)
  • KO'd and attacked (coupe-de-grace)
  • 4 or more Fear tokens
  • Exit battlefield as Disordered/Panicked
  • Assembly Bottled (failed Bottle Test)
```

### Revive
```
Revive — KO'd character may attempt FOR Test to revive.
  • Success: remove KO'd marker, retain Wound tokens.
  • Failure: remains KO'd.
```

## Full QSR Text

### Cover Types
```
Cover — Terrain can provide protection from attacks. There are several types
        of Cover:

Direct Cover — Terrain between the Active model and the target that obscures
               the target.
               Penalize -1 Base die Attacker Hit or Detect Test.

Intervening Cover — Any terrain element that partially obscures the target.
                    This includes any terrain that serves as Direct Cover for
                    the Attacker.
                    LOF to a target which crosses the base of a Distracted
                    model always cause Intervening Cover.
                    Penalize -1 Modifier die Attacker Hit or Detect Test.

Hard Cover — Terrain that provides substantial protection (Buildings, Bunkers,
             Fortifications).
             Penalize -1 Wild die Attacker Damage Test.
```

### Cover Determination
```
A target is behind Cover if, from the perspective of an Opposing model, it is
partially obscured by about half of its visible area by one or more terrain
elements. It must be closer and of same SIZ or lower than any Opposing model
if this Cover affects only base-height.
```

### Terrain Types
```
Clear — No movement penalty, no Cover.

Rough — Degraded terrain. May provide light Cover.

Difficult — Significant movement penalty. May provide Cover.

Obstacle — Blocks movement and LOS. Provides Cover.

Impassable — Cannot be traversed. Blocks LOS. Provides Hard Cover.
```

### Special Cover Rules
```
Open doors and any doorways are Clear. If these are of half base-height or
half base-diameter or smaller, then regard them as Rough.

Open windows are Difficult, but Impassable if smaller than half base-diameter
or base-height.

Models may move through as crouched below low-ceiling obstacles no lower than
to roughly half their base-height as degraded terrain.
```

## Full QSR Text

### Visibility OR
```
Visibility OR — The maximum distance at which models can be clearly seen
                and targeted. Varies by lighting conditions:
                - Day Clear: 16 MU
                - Day Rain/Fog: 8 MU
                - Twilight: 8 MU
                - Night: 4 MU
```

### Line of Sight (LOS)
```
Line of Sight [LOS] — An imaginary straight line from the Active model's
                      visible area to the target's visible area.
                      The visible area of a model is its base-width by its
                      base-height.

                      LOS is blocked by terrain elements that are:
                      - Impassable
                      - Obstacle
                      - Larger than half the model's base-height
```

### Cover
```
Cover — A target is behind Cover if, from the perspective of an Opposing
        model, it is partially obscured by about half of its visible area
        by one or more terrain elements.

        Direct Cover — Terrain between the Active model and the target that
                       obscures the target.
                       Penalize -1 Base die Attacker Hit or Detect Test.

        Intervening Cover — Any terrain element that partially obscures
                            the target. This includes any terrain that
                            serves as Direct Cover for the Attacker.
                            LOF to a target which crosses the base of a
                            Distracted model always cause Intervening Cover.
                            Penalize -1 Modifier die Attacker Hit or Detect Test.
```

### Cohesion
```
Cohesion — Models must remain within Cohesion distance of their Assembly
           members. Cohesion distance = Visibility OR / 2.
```

## Full QSR Text

### Pre-game
```
Use the standard rules. There are just 2 Sides.

Models per Side        BP per Side      Battlefield      End-game
Small   4 to 8         250 to 500 BP    24 × 24 MU       Turn 4
Medium  6 to 12        500 to 750 BP    36 × 36 MU       Turn 6
Large   8 to 12        750 to 1000 BP   48 × 48 MU       Turn 8
```

### Battlefield
```
Terrain.
Set up the Battlefield using the rules for Environments & Terrain.
No special terrain requirements.
```

### Deployment
```
Use standard opposing edge deployment. No Hidden starts unless behind Cover
and out of LOS. Models must deploy within 6 MU of their battlefield edge.
```

### Special Rules
```
None.
```

### Game Length
```
Use the standard game length rules.
```

### Keys to Victory
```
Elimination — +1 VP to Side with highest total BP value of KO'd and
              Eliminated Opposing models at game end

Bottled — +1 VP if Opposing Side fails Bottle Test or has no Ordered models
          remaining at game end

Outnumbered — +1 VP to Side outnumbered 3:2 models or greater at game start;
              +2 VP if outnumbered 2:1 or greater (only if multiple Sides
              remain at game end)
```

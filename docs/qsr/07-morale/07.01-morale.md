## Full QSR Text

### Fear Tests
```
Fear Tests — A type of Morale Test. When failed they cause a character
             to gain Fear tokens and makes them become Nervous, Disordered,
             Panicked, or may even Eliminate that character.

             Required at end of Action upon receiving a Wound token.
             Required if Free or Distracted, and a Friendly model within
             Cohesion becomes either KO'd or Eliminated when not already KO'd.
             Fear Tests are not required if a character is already Disordered
             or if Engaged, unless Distracted.
             Characters are never required to perform more than one Fear Test
             per Turn.

             After performing the Fear Test, if there are cascades, the target
             will acquire Fear tokens:
             - If greater than how many Fear tokens the target currently has,
               it now has as many Fear tokens as misses.
             - Otherwise, the target acquires 1 Fear token.
```

### Nervous/Disordered/Panicked
```
Nervous — If the character has at least 1 Fear token it is Nervous.
          No compulsory action is required.

Disordered — If the character has at least 2 Fear tokens it is Disordered.
             It is no longer Ordered.
             When activated, it must first spend 1 AP on the Compulsory Action
             list. It must use Pushing if needed.
             Safety for this character is defined as the nearest Cover or until
             out of LOS, that is not within 2 AP Movement of any Opposing models
             using their abilities.

Panicked — A character with 3 Fear tokens is Panicked and is regarded as
           Disordered. It is no longer Ordered.
           When activated, it must first spend 2 AP on the Compulsory Action
           list. It must use Pushing if needed.
           Safety for this character is defined as exiting the nearest Friendly
           battlefield entry edge as long as not moving towards any Attentive
           or Ordered Opposing character. Any other edge is otherwise acceptable.
           Panicked characters never cause Opposing models to be Engaged.
```

### Eliminated by Fear
```
Eliminated — A character is Eliminated when:
             - Whenever it receives 4 or more Fear tokens.
             - Whenever they exit the battlefield as a result of being
               Disordered or Panicked.
```

### Rally Tests
```
Rally — Pay 1 AP. If Free, a character may perform an Unopposed POW "Rally Test",
        for itself or a Free Friendly model within Cohesion.
        Upon pass, remove one Fear token for each cascade.
        Characters may only benefit from Rally once per Turn.

        "Friendly" — If an Attentive Ordered Friendly model in Cohesion
                     receive +1m.
        "Safety" — Receive +1w if behind Cover or out of LOS, and not within
                   2 AP Movement of Opposing models.
```

### Breakpoint Morale and Bottle Tests
```
Breakpoint Morale — When an Assembly has half or more of its models KO'd or
                    Eliminated. Double Breakpoint is when it is down to a
                    quarter or less.

Bottle Test — Required of a player at the end of every Turn for each Assembly
              that has reached Breakpoint Morale.
              Pick an Ordered character to perform a Morale Test; this is an
              Unopposed POW Test.
              Apply DR 1 if at Double Breakpoint. Apply another DR 1 if
              outnumbered 2:1 or more total Opposing model count.
              Upon failure or if that Assembly has no Ordered characters the
              game ends for it immediately. Remove all of its models from play;
              they have been Eliminated as a result of having been "bottled".
              They have run away!
              If no Opposing Assemblies remain In-play, the game ends.
              Affected players must perform the Bottle Test. It is possible for
              multiple Assemblies to "Bottle Out".

Forfeit — Instead of performing a "Bottle Test", a player may always decide to
          forfeit and automatically fail.
```

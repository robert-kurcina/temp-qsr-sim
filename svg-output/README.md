# SVG Output

This directory contains static battlefield visualizations generated from the terrain, LOS/LOF, and pathfinding utilities.

## Index

Open `svg-output/index.html` to browse all renders from a single page.

## Notes

- SVGs include embedded layer toggles (grid, Delaunay mesh, terrain, models, paths, vectors, LOS, LOF).
- The index page loads SVGs into an embedded frame for inspection.
